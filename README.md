# R305-Fingerprint-authentication
Interfacing R305 fingerprint sensor with LM3S608 ARM-Cortex-M3 micro controller

It has driver for 
      -16x2 Character LCD
      -SD card with SPI interface
      -DC motor control
      -R305 Finger print module 
      -SIM900 GSM module
     
Compiler is Keil ARM
